## Environment setup

Instructions on how to setup a React Native development environment can be found here: https://reactnative.dev/docs/environment-setup

Make sure to follow the instructions for React Native CLI.

## Installation

```
yarn
```

### iOS installation

```
cd ios
pod install
```

## Start iOS

```
yarn ios
```

## Start android

```
yarn android
```
